We chose a interactive D3 template from bl.ocks.org. We incorporated our dataset with this template. It's a D3 bar chart with tooltips.


For this group project, the work was split evenly. We created a repository on GitHub to share the work. The development process consisted of modifying the fullPage template as well as the D3 chart template. We pushed changes to repository. We also spent time researching additional interaction tools and techniques.
